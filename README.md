# Presentation in Minutes

Create professional HTML presentations instantly using AI.

## Quick Start

This repository contains `AGENT_INSTRUCTIONS.md` - a comprehensive prompt that teaches AI assistants how to generate standalone HTML presentations with PowerPoint-style design.

### How to Use

#### ChatGPT / Claude / Other Chat Interfaces

1. Copy the entire contents of [AGENT_INSTRUCTIONS.md](AGENT_INSTRUCTIONS.md)
2. Paste it as your first message in a new conversation
3. Follow up with your presentation request, e.g.:
   > "Create a presentation about renewable energy for a business audience"
The AI will produce an HTML code block
4. Copy the HTML code into a local `.html` file and open it in a browser (e.g. Chrome)

#### GitHub Copilot: Agent in this Repository
1. Clone this Repo
2. simply ask Copilot in agent mode to create a presentation and it will discover the instructions automatically

#### GitHub Copilot: Custom Presentation Agent
1. Copy the entire contents of [AGENT_INSTRUCTIONS.md](AGENT_INSTRUCTIONS.md)
2. Open Settings (`Ctrl+Shift+P` / `Cmd+Shift+P`) > `Chat: Configure Custom Agents ...` > `Create new Custom Agent`
3. Give it a Name and the necessary tools
4. Paste the the entire contents of [AGENT_INSTRUCTIONS.md](AGENT_INSTRUCTIONS.md) as instructions.


#### Cursor: Agent in this Repository
1. Clone this Repo
2. simply ask Copilot in agent mode to create a presentation and it will discover the instructions automatically

#### Cursor: Custom Rules
1. Open the repository in Cursor
2. Add `AGENT_INSTRUCTIONS.md` to your project rules or paste into chat
3. Use Composer or Chat to request presentations:
   > "Following the presentation framework, create a presentation about [your topic]"

### What You Get

- Single standalone HTML file (no dependencies except Chart.js CDN)
- Professional styling with animations
- Keyboard navigation, touch support, laser pointer
- Print/PDF export ready
- Mobile responsive

### Example Request

> "Create a 6-slide presentation about cloud computing benefits for IT managers. Include a comparison table and a chart showing cost savings."

The AI will generate a complete, ready-to-use HTML presentation.
